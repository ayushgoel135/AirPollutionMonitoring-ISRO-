# Air Pollution Monitoring from Space

Interactive web application for monitoring air pollution across India using satellite data, ground measurements, and machine learning.

## Features

- 3D visualization of air quality data using Three.js
- Satellite data integration (INSAT-3D/3DR/3DS)
- Ground measurement data from CPCB
- Machine learning predictions (Random Forest)
- Interactive maps with zoom from country to district level
